In the "timeCalib" directory,

To compile:

$ make


Prerequisite:

$ mkdir datafile
$ mkdir pdf_files

Then save root files generated by $HOME/cosmicFPGAdecoder/tupleDecoder in main daq (e1039daq@e1039daq1) in "datafile" directory in format of "run_<runnumber>.root"


To run:

$ ./timeCalib <output file tag name> <runnumber, can be multiple runs>

Example:

$ ./timeCalib abc 2067 2068

will analyze "datafile/run_2067.root" and "datafile/run_2068.root" together, then save plots as "pdf_files/c_tdc_abc.pdf"
